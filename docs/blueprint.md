# **App Name**: Urban Nest

## Core Features:

- Property Listings: Allow landlords to post apartments with detailed descriptions, images, and virtual tours.
- Search and Filter: Enable tenants to search for apartments based on location, proximity to schools, price range, property type, and other criteria relevant to students.
- User Accounts: Allow both landlords and tenants to create accounts and manage their profiles. Collect all necessary data during sign-up to minimize errors.
- Contact Agents: Enable tenants to directly contact listing agents or property managers through the platform.
- Apartment Occupancy Management: When a student becomes a tenant of an apartment, the apartment is removed from the available listings.
- Apartment Request Management: Landlords can edit, delete, kick out tenants (if rent is due, or with compensation fee informed via message - not a form), accept requests, or decline requests on the single apartment page.
- Apartment Interaction: On a single apartment page, students can send requests to landlords, review, rate, message landlords, get landlord contacts, share links of apartments to friends on any social app, and bookmark apartments.
- Tenant Apartment Management: Tenants on a single apartment page can renew rent, renew lease, communicate with landlord on maintenance issues, rate and review apartment, decline request to renew rent or lease and see time left to vacate the apartment if landlord request they do.
- Apartment Rules Display: Students, when making a request for an apartment, will also be informed of the rules of the apartment before completing the request form.
- Lease Agreement: Both landlord will sign a form by name which will become a doc they both will download after landlord accepts request to be a tenant under landlord.

## Style Guidelines:

- Primary color: Muted blue (#5D9EBC), evoking trust and stability.
- Background color: Light, desaturated blue (#DDEBF1).
- Accent color: Warm yellow (#F4D35E) to highlight key actions.
- Body font: 'Inter', a sans-serif font providing a modern and clean aesthetic for readability.
- Headline font: 'Space Grotesk', sans-serif to match the feel and styling of the Inter body font.
- Use clear, geometric icons to represent property types, amenities, and search filters.
- Clean, grid-based layout with ample white space to showcase properties and information effectively.
